Tutorial Upload Github via GUI

  1. klik sourche code
  2. klik inisialize repo
  3. pindahin semua file yg ingin di upload ke stanged changes, dengan klik tanda +
  4. masukan pesan commit di kolom atas
  5. dan klik tombol commit
  6. klik publish branch

Bikin Branch baru dengan klik tombol master di pojok kiri bawah, dan pilih create new branch
